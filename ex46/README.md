# 使用這個骨架

1. 拷貝這份骨架目錄，把名字改成你新專案的名字。
2. 再將 NAME模組和 NAME.rb 更名為你需要的名字，它可以是你專案的名字，當然別的名字也行。
3. 編輯你的 NAME.gemspec 檔案，讓它包含你新專案的相關資訊。
4. 重命名 test/test_NAME.rb，讓它的名字匹配到你模組的名字。
5. 開始寫程式吧。
